# Kanban-minimaliste

A minimalist Kanban board app.

- **Frontend** (`Kanban-minimaliste/`) — Angular
- **Backend** (`Kanban-minimaliste-api/`) — Laravel + SQLite

## Run locally

**Backend**
```bash
cd Kanban-minimaliste-api
php artisan serve
```

**Frontend**
```bash
cd Kanban-minimaliste
ng serve
```

Then open: [http://localhost:4200/board](http://localhost:4200/board)
